# ⚛️ React JSX & Components Practice Project

This repository is created as part of my **React learning journey**, where I practice how JSX works, how components are created, and how HTML tags are used inside React applications.

---

## 📌 About This Project

This is a **beginner practice project** focused on understanding the core concepts of React:

- How to create React components
- How JSX works behind the scenes
- How HTML tags are written inside JSX
- How multiple components are combined in a single app
- How data is rendered dynamically inside JSX

This project is purely for **learning and practice purposes**.

---

## 🎯 Learning Goals

Through this project, I learned:

### 🔹 JSX Basics
- Writing HTML-like syntax inside JavaScript
- How JSX gets converted into JavaScript
- Embedding JavaScript expressions inside `{}`

### 🔹 Components
- Creating functional components
- Reusing components inside other components
- Understanding component structure

### 🔹 Rendering Elements
- Rendering HTML tags inside React
- Returning multiple elements using JSX
- Combining multiple components in `App.jsx`

  - `Hello`
