# React Calculator

A simple calculator application built with React that performs basic arithmetic operations. This project was created to practice React fundamentals such as components, props, state management, and event handling.

## Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Clear display using the **C** button
* Decimal number support
* Responsive button layout
* Read-only display screen

## Technologies Used

* React
* JavaScript (ES6+)
* CSS Modules
* Vite

## Getting Started

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```

4. Open the application in your browser.
